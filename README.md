# Agencia de Cuidados de Enfermería a Domicilio

Landing page profesional para servicios de atención médica domiciliaria, desarrollada para mi diplomado de Front-End.

## Autora
Jessenia

## 🛠️ Tecnologías utilizadas
Para el desarrollo de este proyecto se utilizaron las siguientes tecnologías:
* **HTML5:** Estructura de la página web.
* **CSS3:** Estilos personalizados.
* **Bootstrap 5:** Framework para diseño responsivo.
* **JavaScript:** Funcionalidad básica de la plantilla.
* **Font Awesome:** Iconografía del sitio.

## 🚀 Instalación y Despliegue
Para ver este proyecto en tu computadora local, sigue estos pasos:

1. Clona este repositorio usando tu terminal:
   `git clone https://github.com/jesseniado-beep/agencia-enfermeria.git`
2. Entra a la carpeta del proyecto.
3. Abre el archivo `index.html` en tu navegador web.
